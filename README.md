# Spam Mail Prediction using Machine Learning

##  Author
**Aryanshrivastava2512**

---

##  Project Overview

This project aims to classify emails as Spam or Not Spam using Machine Learning classification algorithms.

Spam detection helps in filtering unwanted and potentially harmful emails automatically.

This project demonstrates:
- Text preprocessing
- Feature extraction using NLP techniques
- Model training and evaluation
- Performance comparison

---

##  Dataset Information

The dataset contains:

- Email text messages
- Label (Target Variable):
  - 0 → Not Spam
  - 1 → Spam
